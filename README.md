# Object Detection and Tracking

## 📌 Project Overview
This project is an AI-based Object Detection and Tracking System developed using Python, OpenCV, and YOLOv8.

The system detects and tracks objects in real time using a webcam or video file.

---

## 🚀 Features
- Real-time object detection
- Object tracking with IDs
- Webcam support
- Video file support
- Bounding boxes and labels
- AI-based detection using YOLOv8

---

## 🛠 Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics
- PyCharm

## 📸 Output
The system opens webcam/video and detects objects like:
- Person
- Mobile Phone
- Bottle
- Chair
- Laptop

with bounding boxes and tracking IDs.
